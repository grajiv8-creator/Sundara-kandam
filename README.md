# சுந்தர காண்டம் · Sundara Kandam

A beautiful AI-powered daily reading app for Kamba Ramayanam's Sundara Kandam.
Built for families — each member tracks their own progress independently.

---

## ✨ Features

- Tamil script + English meaning for every verse (AI-powered)
- 5 Padalams · 274 verses total
- Individual progress tracking per family member
- Works on mobile, tablet, desktop
- No login required — just enter your name

## 🚀 Deploy to Vercel (Free — 10 minutes)

### Step 1 — Create free accounts

1. Go to **https://github.com** → Sign Up
2. Go to **https://vercel.com** → Sign Up with GitHub

### Step 2 — Upload to GitHub

1. Go to **https://github.com/new**
2. Repository name: `sundara-kandam`
3. Keep it Public → Click **Create repository**
4. Click **"uploading an existing file"**
5. Unzip this folder → Drag ALL the files inside into the browser
6. Scroll down → Click **Commit changes**

### Step 3 — Deploy on Vercel

1. Go to **https://vercel.com/dashboard**
2. Click **Add New → Project**
3. Find `sundara-kandam` → Click **Import**
4. Leave all settings as default → Click **Deploy**
5. Wait ~60 seconds → 🎉 Your app is live!

### Step 4 — Share with family

Copy the URL (e.g. `sundara-kandam.vercel.app`) and send it to your family via WhatsApp or text.

Everyone opens the same link. Each person enters their name the first time.
Progress is saved on each person's device.

## 📖 How to use

1. Open the app → Enter your name
2. Tap a Padalam number (1–5) to jump to that section
3. Read the Tamil verse and English meaning
4. Tap **Mark as Read** when done
5. Use **Next Unread →** to continue where you left off
6. Check **Progress** tab to see your reading map
7. Check **Family** tab to see everyone's progress

## 🛕 About Sundara Kandam

Sundara Kandam is the 5th chapter (Kandam) of Kamba Ramayanam, the Tamil retelling
of the Ramayana by the poet Kambar (12th century CE). It describes Hanuman's journey
across the ocean to Lanka, his search for Sita, and his triumphant return to Rama.

The 5 Padalams:
1. கடல் தாவு படலம் — Crossing the Ocean (52 verses)
2. இலங்கை காண் படலம் — Seeing Lanka (44 verses)
3. சீதை தேடு படலம் — Searching for Sita (68 verses)
4. தூது படலம் — The Embassy (71 verses)
5. நகர் எரி படலம் — Burning of Lanka (39 verses)
